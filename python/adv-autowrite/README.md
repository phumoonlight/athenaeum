### Environment

- python `3.10.1`
- pip `21.3.1`
- virtualenv `20.13.0`

### Development

- create virtual environment

```bash
virtualenv venv
```
- activate virtual environment

```bash
source ./venv/bin/activate
```

- install dependencies

```bash
pip install -r requirements.txt
```

- start

```bash
python main.py
```
